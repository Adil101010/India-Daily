Banner Advertising Specifications

Leaderboard Banner

  • Dimensions: 728 × 90 pixels
  • Maximum File Size: Under 100KB
  • Frame Rate: 12–16 fps
  • Animation Limit: 15 seconds (animation must stop before 15s)
  • Looping: Not allowed (Replay button permitted)
  • User-Initiated Sound: Allowed
  • Backup Image: Mandatory
  • Branding Requirement: Last frame must include the company’s branding

Accepted File Formats

  • PNG, JPEG, GIF, HTML5

Third-Party Tags

If you are submitting third-party tags, ensure all creative images include visible borders and meet the size & performance specs above.

HTML Advertising Guidelines

  • An HTML document is mandatory for HTML creatives.
  • CSS files must be separate from the HTML.
  • JavaScript files must be separate from the HTML.
  • Images: PSD, JPG, PNG, GIF.
  • Videos (if used inside ad): MP4, OGV, WEBM.
  • If using Google Web Designer (GWD), provide files in open format (not published zip).
Important: Keep creatives lightweight and optimized for fast delivery. Provide a static backup image (PNG/JPG) for all animated ads.
